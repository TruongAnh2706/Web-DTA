# Ứng dụng Web DTA Studio

Dự án này là một ứng dụng web được xây dựng cho DTA Studio sử dụng Vite, React và TypeScript.

## Bắt đầu

### Yêu cầu tiên quyết

- Đã cài đặt Node.js & npm

### Cài đặt

1.  Clone repository:
    ```bash
    git clone <YOUR_GIT_URL>
    ```

2.  Di chuyển vào thư mục dự án:
    ```bash
    cd Web-DTA
    ```

3.  Cài đặt các gói phụ thuộc (dependencies):
    ```bash
    npm install
    ```

4.  Khởi chạy máy chủ phát triển:
    ```bash
    npm run dev
    ```

## Kịch bản (Scripts)

- `npm run dev`: Khởi chạy máy chủ phát triển.
- `npm run build`: Xây dựng ứng dụng cho môi trường production.
- `npm run preview`: Xem trước bản build production trên máy trạm.
- `npm run lint`: Chạy ESLint để kiểm tra mã nguồn.

## Công nghệ sử dụng

- [Vite](https://vitejs.dev/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
